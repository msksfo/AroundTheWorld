About This Project

This project is personal as much as it is professional. I love traveling. I love inspiring travel quotes. I love photography. So I've tied them all together into one beautiful (I hope) collection. It was so hard to reduce a continent to only six images, but I felt that was a good number, without asking too much of the viewer. All of these photos were taken either by myself or by my husband. No, I haven't been everywhere. But it's on my list 😉

The project primarily uses HTML5 and CSS3, although there is a bit of javascript in use for devices smaller than 660 pixels. I've also used the javascript library Lightbox, by Lokesh Dhakar. This enables the user to click an image for a larger view. They can then scroll through the images at this larger size if they choose. On devices smaller than 660 pixels, the gallery sections are initially collapsed so that the user is not forced to scroll forever. They can see the names of the continents and click the continent icon to expand the section.
